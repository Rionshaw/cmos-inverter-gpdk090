# cmos-inverter-gpdk090
CMOS inverter implemented in gpdk090 technology with full design flow: schematic, layout, DC and Transient simulation, delay and power analysis using Cadence Virtuoso.
##Objectives
*Design a CMOS inverter using gpdk090 technology
*Perform schematic and layout design in Cadence Virtuoso
*Analyze -
**Voltage Transfer Characteristics (VTC)
**Transient response
**Propagation delay (tpdr, tpdf)
**Energy consumption
