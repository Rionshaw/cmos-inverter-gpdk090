# cmos-inverter-gpdk090
CMOS inverter implemented in gpdk090 technology with full design flow: schematic, layout, DC and Transient simulation, delay and power analysis using Cadence Virtuoso.<br>
## Objectives<br>
*Design a CMOS inverter using gpdk090 technology<br>
*Perform schematic and layout design in Cadence Virtuoso<br>
*Analyze -<br>
**Voltage Transfer Characteristics (VTC)<br>
**Transient response<br>
**Propagation delay (tpdr, tpdf)<br>
**Energy consumption<br>
